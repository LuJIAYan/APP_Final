## <a>产品方向</a>
做一个提高演讲能力的APP。它私人演讲教练的角色，可以帮助言语恐惧症患者或者不善表达者训练在公众面前演讲的能力。好的演讲与口才能力是练出来的！
### <a>产品定位</a>
目前市面上的演讲APP以看演讲、听演讲，还有PPT演讲稿观点分享较多。真正让演讲者去练习的APP较少。所以，我们的产品定位是做一个让用户去刻意练习演讲的平台。通过APP的各个页面练演讲、学演讲、练思辨组织，多方面提高演讲能力。


### <a>产品核心目标</a>
通过APP的各个页面练演讲、学演讲、练思辨组织，多方面提高演讲能力。

### <a>产品核心功能</a>
#### <a>产品核心立足点</a>
页面 | 重要性 | 可行性| 急迫性
---|---|---|---
 模拟演讲室|预备演讲，熟悉演讲环境|需要VR眼镜辅助，眼镜通过金币兑换|一般，市面上有类似功能
 小游戏PK|提供图片、词语串联故事|提供平台锻炼思考、语言组织、讲故事能力|重要，市面上没有这样的功能
 个人演讲室|提供TED演讲片段跟读|语音识别纠正口头语学习别人如何讲|重要，市面上还没有这样的APP功能
 演讲视频干货观看|学习别人演讲技巧|部分视频可能有版权问题，需要TED平台合作|一般，市面上有类似功能

#### <a>游戏化思维</a>
1. 使用pk的形式去应用练习思辨能力，提高演讲的逻辑性
2. 用金币机制带给用户奖励，提高用户积极性
3. 金币同时可以应用在其他页面，增加页面关联性和产品逻辑性，产品的一致性

#### <a>平静技术原则</a>
1. 使用遵守社会规范：虚拟演讲室设有演讲计时、是VR技术的应用
2. 可以交流但不需要说话：金币商城设有客服交流对话
3. 设备有效利用注意范围边缘： 演讲结束有鼓掌声；虚拟演讲室中，演讲时右上角有时间计时器，若是选择了演讲时长便有倒计时，在剩余30s时提醒一次，剩余十秒时每隔一秒提醒一次，都是通过闪烁的方式提醒用户剩余时间。
4. 放大自身及人类优势：该APP间促进了用户交流，提高用户演讲能力
5. 尽可能减少设备所需注意力：该APP的个人演讲室有评级功能，可以让用户直观地看到演讲能力变化，有针对性提高演讲能力
 
## <a>目标市场</a>
1. 移动端用户（互联网技术的发展，5G时代的到来、流量、便携）
2. 语言沟通类市场（英语流利说，片段跟读模仿纠正）
3. 演讲练习APP市场

## <a>竞品分析</a>
### <a>直接竞品</a>
产品名称 | 优点 |不足
---|---|---
likeso | 有评级、录音、纠错功能|界面过于简单，可能让用户依赖线上演讲
虚拟演讲APP | 模拟演讲场景|功能相对单一需要VR眼镜辅助

### <a>间接竞品</a>

产品名称 | 优点
-|-
英语流利说 | 摘选英语段落，让你录音，模仿练习口语
唱吧 | 创建个人k房，个人练歌，歌曲评级

## <a>用户分析  </a>
1. 不善表达者
2. 想提高自己的演讲能力者
3. 上场会怯场者
4. 恐语症患者

### <a>痛点</a>
1. 演讲紧张，没有预备心理
2. 演讲逻辑不清晰
3. 演讲口头语太多，显得不自信
4. 演讲没有气场
